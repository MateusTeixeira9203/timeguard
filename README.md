# TimeGuard ⏱️

Sistema de controle de tempo por tarefas.

## 📌 Funcionalidades atuais

- Criação de banco SQLite
- Cadastro de tarefas
- Início e parada de timer
- Cálculo automático de duração

## 🚀 Tecnologias

- Python
- SQLite

## ▶️ Como rodar

Criar banco:
py scripts/create_db.py

Testar banco:
py scripts/test_db.py

Testar timer:
py scripts/timer_demo.py
